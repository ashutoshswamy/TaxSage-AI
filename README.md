# TaxSage AI

TaxSage AI is an intelligent tax planning application designed to assist Indian taxpayers in optimizing their tax strategies. Leveraging AI, it provides personalized suggestions, comparisons between tax regimes, and helpful calculators.

## Core Features

- **AI Tax Suggestions**: Utilizes AI (powered by Google Gemini) to analyze user-provided income, deductions, and investment data, offering personalized tax-saving strategies, comparisons between the Old and New Tax Regimes (FY 2024-25), and warnings about common mistakes.
- **Tax Information Pages**:
  - Interactive tax slab viewer for both Old and New Regimes (FY 2024-25).
  - Surcharge and Short-Term Capital Gains (STCG) information page with a surcharge rate calculator.
  - Deductions guide covering Section 80C, Section 80D (Health Insurance), and HRA Exemption, complete with calculators.
- **Report Generation & History**:
  - Generate comprehensive tax reports based on user inputs and AI analysis.
  - Calculate estimated tax payable under both regimes.
  - Export reports as PDF documents.
  - Securely save and access previously generated reports via the Report History page (for logged-in users).
- **Authentication**: Secure user login and signup functionality using Supabase Auth. Protected routes ensure only logged-in users can access sensitive features like report generation and history.
